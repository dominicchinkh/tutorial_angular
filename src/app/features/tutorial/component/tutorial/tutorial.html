Angular Tutorial Overview

Signal

Demonstrates Angular's reactive signals ecosystem with signal(), computed(), advanced linkedSignal() patterns, and resource() for async data loading.

Showcases: signal equality functions, computed derivation, linkedSignal with previous-state preservation, resourceFromSnapshots composition, loading/error handling, template integration

Effect

Demonstrates Angular effect() and afterRenderEffect() for reactive side effects, including browser-safe localStorage sync, cleanup handling, and DOM-phase callbacks after render.

Showcases: effect(), onCleanup(), untracked(), async-boundary behavior, afterRenderEffect phases (earlyRead/write/mixedReadWrite/read), isPlatformBrowser checks

Selector

Demonstrates advanced component selector patterns including element selectors, attribute selectors with [attr], the :not pseudo-class, combining selectors, and comma-separated multi-selectors.

Showcases: element selectors, attribute selectors, :not pseudo-class, combining selectors, multi-selectors

Property Binding

Shows how to bind component properties to DOM properties using [property] syntax. Demonstrates that inputs, host bindings, outputs, and lifecycle methods are inherited when extending a component class, including explicit super() calls for constructor dependencies and lifecycle overrides.

Showcases: one-way binding from component to template, component inheritance (inputs, host bindings, lifecycle methods), super() calls

Event Handling

Demonstrates event binding with (event) syntax to handle user interactions like clicks and keyboard input.

Showcases: click events, keyboard events, event handlers

Template Variable

Demonstrates template reference variables using #variable to capture element values and pass them to component methods.

Showcases: template references, event arguments, value extraction

Reactive Forms

Implements form handling using Angular's ReactiveFormsModule with FormControl and FormGroup for programmatic form management.

Showcases: FormControl, FormGroup, Validators, form validation

Template-Driven Forms

Demonstrates form handling using FormsModule with two-way binding and ngModel for template-driven forms.

Showcases: ngModel, two-way binding, form submission

Input & Output

Shows component communication using input(), output(), and model() signals. Input demonstrates transform functions, booleanAttribute, alias, and HostAttributeToken. Output uses alias for custom event names. Model shows model.required() for two-way binding and binding plain JavaScript properties to model inputs.

Showcases: input.required(), input transform, booleanAttribute, input alias, HostAttributeToken, output alias, model(), model.required(), two-way binding with [(property)]

Content Projection

Demonstrates content projection using ng-content with select for multi-slot projection, ngProjectAs for slot aliasing, fallback content, contentChild() and contentChildren() for querying projected content, ElementRef read option, InjectionToken-based provider queries, and NgComponentOutlet for dynamic component rendering with ngComponentOutletInputs and ngComponentOutletInjector.

Showcases: ng-content select, ngProjectAs, fallback content, contentChild(), contentChildren(descendants: true), read: ElementRef, ProviderToken queries, NgComponentOutlet, ngComponentOutletInputs, ngComponentOutletInjector, Injector.create(), exportAs

Signal Form

Demonstrates the new @angular/forms/signals API — uses form() with a signal model and FormField for reactive, signal-driven form validation with ChangeDetectionStrategy.OnPush.

Showcases: form(), FormField, required(), email(), submit(), signal-based validation, OnPush change detection

Pipes

Shows built-in pipes like CurrencyPipe, DatePipe, DecimalPipe, JsonPipe, TitleCasePipe and custom pipes for data transformation.

Showcases: pipe usage, custom pipes, pipe chaining

Control Flow

Demonstrates modern Angular control flow syntax for conditionals and loops using @if, @for.

Showcases: conditional rendering, list rendering, control structures

Styling

Shows various styling approaches including inline styles, CSS classes, dynamic style binding, ViewEncapsulation modes (Emulated, ShadowDom, None), and host bindings for properties, attributes, classes, styles, and events on the host element.

Showcases: style binding, dynamic classes, ViewEncapsulation, host bindings (role, aria, class, style, tabIndex, events)

Class Binding

Demonstrates CSS class binding using [class.className] and [ngClass] for dynamic class application.

Showcases: conditional class binding, dynamic styling

Directive

Demonstrates a custom attribute directive that uses input(), signal(), and computed() with host bindings and mouse events to create an interactive hover highlight effect.

Showcases: custom directives, host bindings, host listeners, directive inputs, signal-driven styling

Image Optimization

Shows image optimization using Angular's NgOptimizedImage directive with loader configuration for better performance.

Showcases: image lazy loading, responsive images, image optimization

Lifecycle Hooks

Demonstrates all Angular lifecycle hooks: OnInit, OnChanges, DoCheck, AfterContentInit, AfterContentChecked, AfterViewInit, AfterViewChecked, OnDestroy, plus afterNextRender() with read/write phases, DestroyRef, SimpleChanges for tracking input changes, and DOM manipulation via Renderer2 and ElementRef.

Showcases: component lifecycle, hook execution order, afterNextRender write/read phases, SimpleChanges, DestroyRef, Renderer2, ElementRef, DOM API, cleanup

Injectable Service & View Queries

Shows dependency injection using inject() to consume CarService and CartStore. Demonstrates viewChild() and viewChild.required() for accessing child components, viewChildren() for querying multiple children, and template reference variable queries with viewChild('ref'). Cart state uses signal-based computed totalPrice and totalQuantity.

Showcases: inject(), viewChild(), viewChild.required(), viewChildren(), template ref queries, computed totals, CarDetail child component, DestroyRef cleanup

Deferrable Views

Demonstrates the @defer block for lazy-loading and deferring component initialization for improved performance.

Showcases: deferred loading, performance optimization, lazy rendering

Learning Path

This tutorial covers fundamental to advanced Angular concepts:

  • Fundamentals: Selectors, Signals & Effects (computed, linkedSignal, resource, effect, afterRenderEffect), Property Binding (inheritance, super()), Event Handling, Template Variables
  • Component Communication: Input (transform, alias, HostAttributeToken), Output (alias), Model (two-way binding), Content Projection (contentChild, contentChildren, ngProjectAs, ProviderToken, NgComponentOutlet)
  • Forms: Reactive Forms, Template-Driven Forms, Signal Forms (OnPush)
  • Data Presentation: Pipes, Control Flow, Styling (ViewEncapsulation, host bindings), Directives
  • Advanced: Lifecycle Hooks (afterNextRender, SimpleChanges, DestroyRef, Renderer2, ElementRef), Dependency Injection, View Queries (viewChild, viewChildren), Route Navigation, Performance Optimization
