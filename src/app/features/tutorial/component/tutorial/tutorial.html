Angular Tutorial Components Overview

Signal

Demonstrates Angular's reactive signals ecosystem with signal(), computed(), linkedSignal(), and resource() for state derivation, writable linked state, and async data loading.

Showcases: signal creation, computed derivation, linked writable state, resource loading/error states, template integration

Effect

Demonstrates Angular's effect() function to run side effects automatically whenever tracked signals change. Shows saving theme preferences to localStorage and logging user activity changes with platform checks for server-side rendering safety.

Showcases: effect(), automatic signal tracking, side effects execution, localStorage persistence, isPlatformBrowser checks

Property Binding

Shows how to bind component properties to DOM properties using [property] syntax.

Showcases: one-way binding from component to template

Event Handling

Demonstrates event binding with (event) syntax to handle user interactions like clicks and keyboard input.

Showcases: click events, keyboard events, event handlers

Template Variable

Demonstrates template reference variables using #variable to capture element values and pass them to component methods.

Showcases: template references, event arguments, value extraction

Reactive Forms

Implements form handling using Angular's ReactiveFormsModule with FormControl and FormGroup for programmatic form management.

Showcases: FormControl, FormGroup, Validators, form validation

Template-Driven Forms

Demonstrates form handling using FormsModule with two-way binding and ngModel for template-driven forms.

Showcases: ngModel, two-way binding, form submission

Input & Output

Shows component communication using @Input() and @Output() decorators for parent-child component interaction.

Showcases: input properties, output events, event emission

Model Input

Demonstrates Angular model() for writable component state alongside input() for one-way data, ideal for form controls and two-way-friendly UI components.

Showcases: model.required(), input(), writable signals, checkbox state toggling

Content Projection

Demonstrates content projection using ng-content to allow parent components to project content into child components.

Showcases: ng-content, component composition

Signal Form

Demonstrates the new @angular/forms/signals API — uses form() with a signal model and FormField for reactive, signal-driven form validation with ChangeDetectionStrategy.OnPush.

Showcases: form(), FormField, required(), email(), submit(), signal-based validation, OnPush change detection

Pipes

Shows built-in pipes like CurrencyPipe, DatePipe, DecimalPipe, JsonPipe, TitleCasePipe and custom pipes for data transformation.

Showcases: pipe usage, custom pipes, pipe chaining

Control Flow

Demonstrates modern Angular control flow syntax for conditionals and loops using @if, @for.

Showcases: conditional rendering, list rendering, control structures

Styling

Shows various styling approaches including inline styles, CSS classes, and dynamic style binding.

Showcases: style binding, dynamic classes, CSS integration

Class Binding

Demonstrates CSS class binding using [class.className] and [ngClass] for dynamic class application.

Showcases: conditional class binding, dynamic styling

Directive

Demonstrates a custom attribute directive that uses input(), signal(), and computed() with host bindings and mouse events to create an interactive hover highlight effect.

Showcases: custom directives, host bindings, host listeners, directive inputs, signal-driven styling

Image Optimization

Shows image optimization using Angular's NgOptimizedImage directive with loader configuration for better performance.

Showcases: image lazy loading, responsive images, image optimization

Lifecycle Hooks

Demonstrates all Angular lifecycle hooks: OnInit, OnChanges, DoCheck, AfterContentInit, AfterContentChecked, AfterViewInit, AfterViewChecked, OnDestroy.

Showcases: component lifecycle, hook execution order, cleanup

Injectable Service

Shows dependency injection using the inject() function to consume a CarService, then renders the car list as navigable links using RouterLink.

Showcases: service injection, inject(), getCars(), RouterLink, route navigation

ViewChild

Demonstrates viewChild() to obtain a direct reference to a child CartDisplay component and call its methods from the parent. Cart state is managed by a signal-based CartStore service with computed totalPrice and totalQuantity, and DestroyRef is used to clean up checkout timers on component destruction.

Showcases: viewChild(), child component method calls, CartStore, signal.asReadonly(), computed totals, DestroyRef cleanup

Deferrable Views

Demonstrates the @defer block for lazy-loading and deferring component initialization for improved performance.

Showcases: deferred loading, performance optimization, lazy rendering

Learning Path

This tutorial covers fundamental to advanced Angular concepts:

  • Fundamentals: Signals (computed, linkedSignal, resource, effect), Property Binding, Event Handling, Template Variables
  • Component Communication: Input/Output, Model Input, Content Projection
  • Forms: Reactive Forms, Template-Driven Forms, Signal Forms
  • Data Presentation: Pipes, Control Flow, Styling, Directives
  • Advanced: Lifecycle Hooks, Dependency Injection, ViewChild, Route Navigation, Performance Optimization
